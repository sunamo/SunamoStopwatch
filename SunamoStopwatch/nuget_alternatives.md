# NuGet Alternatives to SunamoStopwatch

This document lists popular NuGet packages that provide similar functionality to SunamoStopwatch.

## Overview

Performance timing utilities

## Alternative Packages

### System.Diagnostics.Stopwatch
- **NuGet**: System.Diagnostics.Debug
- **Purpose**: Built-in stopwatch
- **Key Features**: High-resolution timing, elapsed time

### BenchmarkDotNet
- **NuGet**: BenchmarkDotNet
- **Purpose**: Benchmarking framework
- **Key Features**: Accurate performance measurements, statistical analysis

### MiniProfiler
- **NuGet**: MiniProfiler
- **Purpose**: Profiling library
- **Key Features**: Web request timing, database profiling

### System.Diagnostics.Activity
- **NuGet**: System.Diagnostics.DiagnosticSource
- **Purpose**: Distributed tracing
- **Key Features**: Activity tracking, correlation

## Comparison Notes

Stopwatch for simple timing. BenchmarkDotNet for accurate benchmarks. MiniProfiler for profiling.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Diagnostics.Stopwatch**: Built-in stopwatch
- **BenchmarkDotNet**: Benchmarking framework
- **MiniProfiler**: Profiling library
